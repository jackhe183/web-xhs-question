# v2 主题测试 - 使用说明

## 📁 文件说明

| 文件名 | 主题 | 配色 | 说明 |
|--------|------|------|------|
| `emotion_test.html` | 情感人格测试 | 橙红色 💕 | 测试你的情感人格类型 |
| `career_test.html` | 职业测试 | 金色 💼 | 测试你的理想职业 |

## 🚀 快速使用

### 方法1：直接打开
直接双击对应的 `.html` 文件，用浏览器打开即可。

### 方法2：创建新主题
1. 复制任意一个现有的 `.html` 文件
2. 重命名为你的主题名称（如 `city_test.html`）
3. 打开文件，找到 `APP_CONFIG` 部分（约第180行）
4. 修改配置：
   - `theme` - 修改主题色
   - `content` - 修改页面文案
   - `questions` - 修改题目
   - `resultTypes` - 修改结果类型

## 📝 APP_CONFIG 配置说明

### 1. 主题配色
```javascript
theme: {
    primaryColor: '#ff6b35',      // 主色（按钮、进度条）
    secondaryColor: '#ee4d2d',    // 辅色（渐变）
    bgStart: '#fff5f0',           // 背景起始色
    bgMid: '#ffe8dc',             // 背景中间色
    bgEnd: '#ffd9c8',             // 背景结束色
    iconEmoji: '💕'               // 封面图标
}
```

### 2. 页面文案
```javascript
content: {
    pageTitle: '页面标题',
    coverTitle: '封面标题',
    coverSubtitle: '封面副标题',
    coverDesc: '封面描述',
    startBtn: '开始按钮文字',
    // ... 其他文案
}
```

### 3. 题目数据
```javascript
questions: [
    {
        q: '问题文字？',
        opts: [
            { t: '选项文字', scores: { A: 3, B: 1, C: 0, D: 0 } },
            { t: '选项文字', scores: { A: 0, B: 3, C: 1, D: 0 } },
            // ... 最多4个选项
        ]
    },
    // ... 更多题目
]
```

### 4. 结果类型
```javascript
resultTypes: {
    "A": {
        typeName: "类型名称",
        tags: ["标签1", "标签2", "标签3"],
        bestMatch: "最佳匹配描述",
        potentialChallenge: "潜在挑战",
        desc: "详细描述文字"
    },
    // ... B、C、D 类型
}
```

## ⚠️ 注意事项

1. **题目数量**：建议 3-20 题，当前示例为 3 题
2. **选项数量**：每题必须是 4 个选项
3. **类型数量**：必须包含 A、B、C、D 四种类型
4. **分数规则**：每个选项的 scores 总和最好接近，避免偏差过大

## 🔧 修改模板时的提示

1. **只修改 APP_CONFIG 内的内容**
2. **不要修改核心引擎代码**（标有"不需要修改"的部分）
3. **保存时选择 UTF-8 编码**，避免中文乱码
4. **修改后用浏览器测试，确保功能正常**
