## 二、快速部署

### 方式1：直接使用（推荐）
1. 打开 `commercial.html` 文件
2. 右键选择"在浏览器中打开"
3. 测试所有功能是否正常

### 方式2：Cloudflare Pages 部署
1. 登录 https://dash.cloudflare.com
2. 进入 Pages > 创建项目 > 连接到 Git
3. 选择仓库后，构建设置留空
4. 部署完成后获得公开链接

### 方式3：Vercel 部署
1. 登录 https://vercel.com
2. 导入 GitHub 仓库
3. 一键部署，自动 HTTPS

A: 购买后会收到测试链接，点击即可在手机/电脑上使用。
